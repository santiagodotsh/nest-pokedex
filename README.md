## Project setup

```bash
$ npm install
```

## Start database

```bash
$ docker compose up
```

## Run SEED

```bash
$ http://localhost:3000/api/v2/seed
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
